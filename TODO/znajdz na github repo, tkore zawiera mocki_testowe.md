# znajdz na github repo, tkore zawiera mocki/testowe wersje popularnych social mediow i stron globalnych

Znalazłem kilka publicznych repozytoriów na GitHubie, które zawierają mocki / testowe wersje popularnych serwisów społecznościowych i globalnych stron — poniżej krótkie opisy i linki.

- Mock Social Media Website — narzędzie open-source do eksperymentów z zachowaniami na social media, przydatne jako testowy interfejs social feedu.[^1]
- MockSocial (fraxxio) — generator realistycznych zrzutów ekranu postów dla Twittera, Instagramu, Facebooka, Discorda i ChatGPT; można klonować i używać do tworzenia przykładowych postów.[^2]
- MockSocial (ashishguleria04) — rozbudowany generator mockupów czatów i sociali (Next.js, Tailwind, wiele skórek platform jak WhatsApp, Instagram, X itd.) z instrukcją uruchomienia lokalnego serwera deweloperskiego.[^3]
- Tematy GitHub: social-media-website / social-media-clone / social-media-mockup — listy repozytoriów-klonów i mocków (przydatne do znalezienia wielu przykładów: klony Facebooka, Twittera/X, Instagram, TikTok i inne).[^4][^5][^6]

Jeśli chcesz, mogę:

- sklonować i przygotować krótką listę konkretnych repo (nazwa + opis + instrukcja uruchomienia) z wybranej kategorii (np. tylko generatory screenshotów albo pełne klony),
- wyszukać repo tylko w języku którym wolisz (Polish/English) lub tylko z konkretnymi technologiami (React, Next.js, Node.js),
- albo przygotować polecane repo do szybkich testów (uruchomienie lokalne + przykładowe polecenia).
